Kanał RSS - informacje

Poniżej znajdziesz informacje na temat naszego kanału RSS i możliwości, jakie on daje swojemu użytkownikowi.

Adres:


Kanał dostępny jest pod bazowym adresem:
[url]

Dostępne formaty danych:


Kanał RSS umożlwia wyświetlenie danych w formatach XML oraz JSON poprzez dodanie odpowiednie rozszerzenia (np. .json) do swojego adresu.
  • xml - domyślny format danych używany dla kanału, w przypadku braku określonego formatu w zapytaniu wyniki zostaną zwrócone w jego strukturze
    • przykład: [url]
  • json - popularny, szczególnie ostatnimi czasy, format JavaScript Object Notation.
    • przykład: [url]

Dostępne parametry:


  • limit - oferuje limitowanie ilości wyników (tutaj wpisów) do podanej wartości.
    • przykład: [url]
  • excludedAuthors - umożlwia wykluczenie poszczególnych autorów z wyników danego zapytania. Kolejni autorzy oddzieleni powinni być przecinkiem.
    • przykład: [url]
  • łączenie parametrów - oczywiście możliwe jest łączenie wspomnianym powyżej parametrów
    • przykład: [url]
